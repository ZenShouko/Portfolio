Opdrachten

Voor ons semester project hebben wij gekozen om een website te maken voor een fictief bedrijf genaamd "Depo Den Opschlagh". Dit bedrijf is een opslag bedrijf die opslagruimtes verhuurt aan klanten. De klanten kunnen op de website een opslagruimte reserveren en betalen. De website is gemaakt met HTML, CSS en JavaScript voor het frontend-gedeelte. Voor het backend-gedeelte is er gebruik gemaakt van C# en SSMS. Origineel hadden wij garages, container en eventueel tuinen in gedachten. Maar uiteindelijk, na wat tijdgebrek en 3 teamleden die het project verlaatten, hebben wij besloten om alleen garages te doen.

Reflectie

Ons project is niet goed verlopen. Er was geen goed samenwerking in ons team, geen eerlijke communicatie en geen duidelijke afspraken. Er ontbrak veel kennis waardoor de takenverdeling niet goed verliep. Niemand stelde zich verantwoordelijk voor een taak/issue.

Ik kreeg de opdracht om de teamgenoten bij te sturen en te helpen met de taken. Maar we kwamen niet vooruit en de deadline kwam steeds dichterbij. Ik heb toen besloten om de taken van de teamgenoten over te nemen. Uiteindelijk begonnen mensen het project te verlaten en hadden wij nog maar een team van 2.

Ik heb een paar pagina's ontworpen (homepage, login en alle product/order gerelateerde pagina's). Ik heb ook de database volledig ontworpen. Ik heb de code geschreven dat gebruiker login behandeld. Alle product en order gerelateerde code is ook door mij geschreven. De volledige wpf applicatie is ook door mij geschreven. De code structuur was veranderd door een teamgenoot, waardoor de code voor de login in de wpf niet meer werkt. De eindpresentatie heb ik ook gemaakt.

Ik heb veel geleerd van dit project. Ik heb geleerd hoe ik een database moet ontwerpen en hoe ik met een database kan communiceren. Een API dat alle verzoeken verwerkt. Maar het meest belangrijke: Ik heb geleerd dat een goed team zeer essentieel is voor een project. Een goede planning en duidelijke afspraken zijn onmisbaar. Een goede communicatie is ook zeer belangrijk. Het team bepaalt de sfeer van het project.

...
Had enorm uitgekeken om te werken in een team en werd ontgoocheld.
